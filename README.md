# HackCanada2026

A fully functional AI native Blender alternative built with Three.js, Meshy.ai, and Gemini. Use AI to help you block out your model, texture, refine, and finalize. 

# Features
- 3D scene with hierarchical structure
- Dockable windows to customize view according to user preference
- AI powered primitive model blocking out
- Image generation and refinement
- Mesh generation using images as references
- AI animation and rigging

# How to Run
1. Clone the repo `git clone https://github.com/evanlambb/HackCanada2026.git`
2. In the root directory of the project install the dependencies using `npm i`
3. Run the project using `npm run dev` and go to localhost port it launches at
